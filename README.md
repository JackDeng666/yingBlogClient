## 嘤之博客客户端项目 
## vue-cli的单页面应用项目

线上地址：https://anotherbug.cn

## Project setup（下载依赖包）
```
npm install
```

### Compiles and hot-reloads for development (以热更新的开发模式启动项目)
```
npm run serve
```

### Compiles and minifies for production (编译打包项目)
```
npm run build
```

- public
  - editor.md 富文本编辑器
  - zoomify 图片放大插件
  - img 静态资源

- utils
  - api  api请求封装文件
  - config 配置文件
  - canvasObject 项目写的各种canvas动画类
